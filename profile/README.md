# Reel-Crew

A group movie coordination app. Create a watch group, assign a film, set a deadline — and track who's watched it and who hasn't.

## The App

ReelCrew lets you and your friends stay on the same page when it comes to watching films together. Browse a film catalog powered by OMDB, build a personal watchlist, create or join a watch group, and coordinate who's seen what — all in one place.

**Built with:** React 19 · React Router 7 · Material UI · Node · Express 5 · PostgreSQL · Vite
